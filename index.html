Introduction

COSÌ is an open online catalog of Italian discourse markers (Segnali discorsivi). It provides a comprehensive, corpus-based resource for the analysis, annotation, and exploration of discourse markers (DMs) in contemporary spoken Italian.

Background and Motivation

COSÌ builds upon previous resources like the Diccionario de Partículas Discursivas del Español (DPDE) but improves upon limitations such as lack of corpus linkage, metadata access, and dynamic annotation exploration.

Rooted in linked data principles, COSÌ emphasizes interoperability, sustainability, and usability with the KIParla corpus.

Annotation Initiatives

The primary model is MDMA (Bolly et al., 2015), focusing on corpus-based, replicable, objective DM annotation. COSÌ enhances this by including clusters, function layers, and speaker profiling.

Building COSÌ

General Architecture

COSÌ consists of:

  • Website: UI for DM exploration (planned).
  • Open Repository: Annotated data, guidelines, substitution tests, and context files.

A Corpus-Based Approach

All data derives from the KIParla corpus and its modules (KIP, KIPasti, ParlaTO, ParlaBO), using random sampling via NoSketchEngine with full metadata retention.

Selecting DMs

  • Based on frequency lists from KIParla and VoLIP.
  • 1,517 tokens analyzed; 175 flagged DMs.
  • Occurrences range from 101 (evidentemente) to 30,666 (eh).
  • High-frequency DMs capped at 800 examples; low-frequency DMs fully included.

Annotation Schema Design

Annotation differentiates between DM and non-DM uses using three substitution diagnostics, with functional labels grounded in literature but adapted for practical use.

Interoperability with the Source Corpus

Annotations preserve KIParla metadata and comply with the CoNLL-U format to ensure integration and longevity across linguistic resources.
